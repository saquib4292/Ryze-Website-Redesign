# Ryze – Website Redesign (Frontend Assignment)

## 📌 Project Overview
This project is a redesign of the **Ryze (get-ryze.ai)** website, created as part of a frontend internship assignment.  
The goal was to improve the website’s **visual appeal, clarity, navigation, and responsiveness**, while keeping the design clean, modern, and professional.

The project focuses only on **frontend design and structure** (no backend or APIs).

---

## 🎯 Objectives
- Redesign the homepage and inner pages
- Improve UI/UX, layout, and readability
- Add clear call-to-action buttons
- Ensure responsive and mobile-friendly design
- Organize the UI into reusable components

---

## 🧩 Pages Implemented
- **Home** – Hero section, value proposition, features overview
- **Features** – Product features and benefits
- **Pricing** – Pricing plans for different user needs

---

## ✨ Key Features
- Clean and modern UI
- Responsive layout (desktop & mobile friendly)
- Smooth navigation using React Router
- Clear Call-To-Actions (Get Started, Book a Demo)
- Modular component-based structure

---

## 🛠️ Tech Stack
- **React (TypeScript)**
- **Vite**
- **React Router DOM**
- **HTML & CSS (inline styling)**
- **JavaScript / TypeScript**
